HR Analytics Dashboards

A collection of Power BI dashboards focused on HR analytics and workforce insights.

## Dashboards Included
- Attrition Dashboard
- FNF Dashboard
- Exit Dashboard
- HR Parametrics Dashboard

## Key Features
- Employee attrition analysis
- Exit and FNF tracking
- Workforce KPIs and HR metrics
- Interactive filters and visualizations

## Tools Used
- Power BI
- DAX
- Excel / SQL / SalesForce / ConnectWise / HRMS 

## Purpose
To provide actionable HR insights for workforce management, employee tracking, and operational reporting.
